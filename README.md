# WIP full-stack web app for planning birdwatching trips
### Features
- Use an interactive map to find local bird finds, a detailed weather forecast, and recreation areas
- Log your favorite birds, and keep track of which ones you have already found
### Backend
- Built with [ASP.NET](https://dotnet.microsoft.com/en-us/apps/aspnet) Core 10 (written in C#)
- Features a RESTful API for retrieving external API data and calling the database 
- Database built with SQL (WIP)
### Frontend
- Frontend built with [Angular](https://angular.dev/) (written in TypeScript)
- Additional tools: [Tailwind](https://tailwindcss.com/), [DaisyUI](https://daisyui.com/)
### Attribution
- External APIs: [Ebird.org](https://ebird.org/home), [National Weather Service](https://www.weather.gov/documentation/services-web-api), [Recreation Information Database](https://ridb.recreation.gov)
- Icons: [Heroicons](https://heroicons.com/), [Feather Icons](https://feathericons.com/)
